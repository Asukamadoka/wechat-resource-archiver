# 微信资源归档 OpenClaw Skill

将微信 ClawBot 中转发的公众号文章、网页、图片、视频、音频、文件和文字卡片归档到本地知识库。

## 一键安装入口

把下面的安装文档链接发给支持读取链接的 OpenClaw / ClawBot，并要求它按照文档安装：

```text
https://raw.githubusercontent.com/Asukamadoka/wechat-resource-archiver/main/wechat-resource-archiver_INSTALL.md
```

也可以打开 [安装指南](wechat-resource-archiver_INSTALL.md) 手动安装。

## 支持内容

- 公众号文章：保存 Markdown、原始 HTML、正文图片和元数据；
- 普通网页：保存 Markdown、原始 HTML、图片和元数据；
- 图片、视频、音频和文档：保存原文件、备注、哈希和元数据；
- 视频号卡片：保存卡片信息；只有 ClawBot 提供实际媒体文件时才归档视频；
- 按 URL 或文件 SHA-256 自动去重。

技能不会绕过微信登录、访问验证、付费限制或 DRM。
